Pattern Box is a WordPress pattern library plugin for the block editor, providing pre-designed patterns for hero and about sections.

== Description ==

Pattern Box enhances your WordPress site with a variety of pre-designed block patterns for the Gutenberg editor. You can easily enable or disable specific patterns and categories through a user-friendly dashboard.

**Features:**
* Hero sections
* About sections
* Team sections
* Easy to enable/disable patterns and categories
* Seamless integration with Gutenberg

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/pattern-box` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to 'Pattern Box' in the WordPress admin menu to configure the settings.

== Frequently Asked Questions ==

= How do I enable or disable specific patterns? =

Go to the 'Pattern Box' settings page in the WordPress admin menu. From there, you can enable or disable categories and individual patterns.

= What are block patterns? =

Block patterns are predefined block layouts that you can insert into your posts and pages. They allow you to create complex designs quickly and easily.

= Can I suggest new patterns to be added? =

Yes! We welcome suggestions. Please visit our [support forum](https://dragwp.com/support) to share your ideas.
